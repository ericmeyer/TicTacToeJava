## Setting up the Application

1. `git clone https://github.com/ericmeyer/TicTacToeJava`
2. `cd <project-directory>`

### Running the Tests

`./gradlew test`

### Running the application

For verbose output: `./gradlew run`
For cleaner output: `./gradlew run -q`
